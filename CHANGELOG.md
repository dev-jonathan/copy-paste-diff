# Change Log

All notable changes to the "copy-paste-diff" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.0] - 2025-11-10

### Added
- Manual diff paste with Ctrl+Alt+V / Cmd+Alt+V shortcut
- Automatic diff mode (optional) for large pastes via Ctrl+V / Cmd+V
- Smart detection to skip diff when only whitespace/indentation changes
- Configuration settings for paste override and minimum paste size
- Native VS Code diff viewer integration
